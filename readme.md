# Учебный проект Django

## Описание проекта

Этот проект создан для практики работы с Django и Bootstrap. Целью проекта было выполнение следующих заданий:

- Создание отдельных файлов для меню и подвала сайта.
- Подключение этих файлов к основному шаблону через include.
- Создание четырёх страниц сайта с использованием основного шаблона.
- Стилизация всех страниц с помощью Bootstrap.

## Выполненные задачи

1. **Создание отдельных файлов для меню и подвала**:
   - Были созданы файлы `menu.html` и `footer.html`, которые содержат код для меню навигации и подвала сайта соответственно.

2. **Подключение меню и подвала к основному шаблону**:
   - В основном шаблоне `base.html` были подключены файлы `menu.html` и `footer.html` с помощью тега `{% include %}`. Это позволило использовать единый код для меню и подвала на всех страницах сайта.

3. **Создание четырёх страниц сайта**:
   - Были созданы страницы `home.html`, `about.html`, `services.html` и `contact.html`. Эти страницы используют основной шаблон `base.html` и содержат уникальный контент для каждой страницы.

4. **Стилизация страниц с помощью Bootstrap**:
   - Все страницы были стилизованы с использованием фреймворка Bootstrap. Это обеспечило единый и современный внешний вид сайта, улучшив пользовательский интерфейс и общее восприятие сайта.

## Заключение

Проект позволяет понять основные принципы работы с шаблонами в Django и показывает, как можно использовать Bootstrap для стилизации веб-страниц. Выполнение этих задач помогает улучшить структуру кода, делая его более модульным и поддерживаемым.
